## Installation: 

This application also requires node.js, file system and the inquirer module (need to run: npm i inquirer@6.5.0). This app uses Jest to run it's testing.

run: <br>
npm install <br> 
npm i inquirer@6.5.0 <br>
npm i jest --save-dev
